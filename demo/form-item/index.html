Phoenix Module (form-item) GitHub

px-enter

Chamar função ao teclar enter.

<input type="text" px-enter="myFunction()"/>
Clique no input e tecle enter {{enterTime}}

px-cpf-mask

Definir máscaras para campos (com base em ui-mask).

<input type="text" ng-model="cpf" px-cpf-mask/>
Use também: px-cnpj-mask, px-br-phone-mask px-ddd="true", px-br-cep-mask, ui-mask (dependencie)

px-number-mask

Definir máscaras para números.

<input type="text" ng-model="cpf" px-number-mask/>

px-message

Mensagens de erros.

<input id="showError" ng-model="showError" required/>
<px-message px-element="showError" />
Expiremente usar tabem : <px-message px-element="showError" ng-if="demo.showError.$touched" />
